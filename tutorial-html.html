HTML basics

HTML is the structure of a webpage. It gives meaning to text, images, links, buttons, and sections.

What it is

HTML stands for HyperText Markup Language. It defines the layout and content of web pages.

Key terms

Tags, elements, attributes, headings, paragraphs, and lists are the building blocks of HTML.

Learn by doing

Try making a heading, paragraph, image, and link in one small page.
